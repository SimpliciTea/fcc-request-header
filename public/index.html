API Basejump: Request Header Parser Microservice

GitHub

Hi there! This is my implementation of FreeCodeCamp's API Basejump 'Request Header Parser Microservice'.

User Stories:

  1. I can get the IP address, language and operating system for my browser.

Example usage:

/api/whoami /1450137600

Example output:

{"ipaddress":"70.199.131.185","language":"en-US","software":"X11; Ubuntu; Linux x86_64; rv:42.0"}
